## 배경

이미 운영 중인 계좌들:
- **위탁**: NPS 미국주식 추적 (미국 비중)
- **ISA**: 배당주 5종목 (한국 가치주)
- **발행어음**: 현금성

IRP 연금은 기존에 8종목(금현물, 고배당, 리츠, S&P500, 인도, 달러, 배당다우존스, 국고채)을 보유 중이었는데, 전부 보수적 자산이라 성장 베팅이 없었습니다.

## 전략 수립 과정

### 테마 검토

네이버 금융 API로 KRX ETF 1,088개 전수조회 후 키워드 필터링:
- AI/인공지능: 77개
- 로봇: 12개
- 반도체: 48개

코인이 1순위였지만 IRP에서 불가능. 중동(두바이/UAE)도 KRX ETF 없음.

## 적립 전략

### Phase 1: 기존 자금 소진 (약 10주)
- 기존 7종목 매도 → 약 200만원 현금
- 매주 월요일 20만원씩 분할 매수

### Phase 2: 신규 적립 (이후 계속)
- 매주 15만원 입금 + 매수
- 50세까지 꾸준히

### 연 6% 보수적 시뮬레이션 (매주 15만원, 20년)
- 원금: 1.56억원
- 예상 자산: **3.04억원**
- 55세 이후 매년 1,200만원씩 수령 가능 (월 100만원, 세금 3.3~5.5%)

## API 자동매매 시도 → 실패

ISA 배당 포트에서 KIS API 자동매매(execute)가 잘 돌아가서, IRP에도 같은 방식을 적용하려 했습니다.

```
에러: (RT_CD: 7, MSG_CD: APBK1744)
퇴직연금계좌는 해당 서비스가 불가합니다.
```

### 조사 결과

KIS API 퇴직연금 계좌 지원 현황:

| 기능 | 지원 |
|------|------|
| 잔고조회 | ✅ |
| 예수금조회 | ✅ |
| 매수가능조회 | ✅ |
| 체결기준잔고 | ✅ |
| 미체결내역 | ✅ |
| **주문(매수/매도)** | **❌** |

퇴직연금 전용 주문 API 자체가 KIS API에 존재하지 않습니다. 법적 규제 때문으로 추정.

## 대안: 반자동 시스템

자동매매는 안 되지만, 계획+추적은 자동화:

```
buy-plan → 텔레그램 알림 "이번주 이거 사세요"
         → 앱에서 직접 매수 (5분)
confirm  → state 업데이트
status   → 비중 체크
```

매주 월요일, 텔레그램으로 종목/수량이 오면 앱 열고 따라 사기만 하면 됩니다.

---

## 오늘의 교훈

1. **모든 계좌가 API 매매를 지원하는 건 아닙니다.** 위탁/ISA는 되지만 퇴직연금은 안 됩니다.
2. **먼저 API 지원 범위를 확인하고 코딩하세요.** 코드 다 짜고 나서 "안 됩니다" 에러 보면 허무합니다.
3. **완전 자동화가 안 되면 반자동으로.** 계획 생성과 기록은 자동, 실행만 수동 — 이것만으로도 충분히 편합니다.

---

## 현재 결론

API 자동매매가 안 되는 상황에서, 한투 앱의 적립식 매수 기능도 제대로 동작하지 않아 **기존 IRP 세팅(8종목)을 당분간 유지**하기로 했습니다.

전략과 코드는 준비해뒀으니:
- KIS API가 퇴직연금 주문을 지원하게 되면 → 자동화 적용
- 수동으로 전환하기로 결정하면 → buy-plan + confirm으로 운영
- 코인 ETF가 KRX에 상장되면 → 비중 조정해서 편입

*이 글은 실제 투자 과정을 기록한 것입니다. 투자 판단은 본인 책임입니다.*
